## Ctrl + B

## Ctrl + A

## Ctrl + E

光标移动到==行尾==



## Ctrl + P

## Ctrl + N

## Ctrl + H

删除光标==前一个字符==

## Ctrl + D

删除光标==所在字符==



## Ctrl + U

删除光标（不包含）至行首全部内容

## Ctrl + K

删除光标（包含）至行尾全部内容

## Ctrl + W

删除光标前的一个单词

## ==Ctrl + Y==
